## 🎨 Customization

### Change Colors (Easy!)
Edit `static/css/custom.css`:
```css
:root {
    --primary-color: #3b82f6; /* Change this! */
}
```

### Update Logo
Replace: `static/images/logo.png`

### Modify Text
Edit the template files in `templates/`

**See [CUSTOMIZATION_GUIDE.md](CUSTOMIZATION_GUIDE.md) for details!**
